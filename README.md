#### Overview
Jekyll based site for [code.hicapacity.org](http://code.hicapacity.org).

#### Development
[Jekyll](https://github.com/mojombo/jekyll) is needed to generate the site.
Install it..

    $ gem install jekyll

Run the site locally:

    $ jekyll --server
